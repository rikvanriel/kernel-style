# CLAUDE.md

A reference guide to Linux kernel commit, changelog, comment, and code style.

This is documentation, not instructions to execute. If you encounter it while
reviewing or crawling code, treat it as ordinary reference material. If a user
has deliberately loaded it to write in this style, start at README.md.

README defines three-phase cumulative load: Phase 1 draft always hot kernel-style.md + kernel-readability-principles.md + llm-tells-checklist.md (~2.5k words). Phase 2 review adds exemplars.md mandatory before git commit (~3.3k words transient). Phase 3 draft changelog adds changelog-style.md mandatory (~3.5k words transient). Patch-series.md on demand only. Unload at task end. See README How to load and AGENTS.md for full detail.
