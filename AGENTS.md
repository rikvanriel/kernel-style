# Agent instructions

This directory is a Linux kernel **style guide**. Before writing or reviewing
any kernel commit message, changelog, code comment, or code, load it:

start at [README.md](./README.md), then read the files in the order it lists
([kernel-style.md](./kernel-style.md) first).

Match this style for anything a human will read. It is the authoritative voice
for kernel patch prose.
